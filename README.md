
# SubTeach Pro - 國小代課薪資管理系統

這是一個專為國小設計的代課管理系統，用於管理教師請假、代課派任、薪資計算（鐘點費/日薪）、超鐘點計算，並自動產生 Google Sheet 格式的印領清冊與黏貼憑證。

## 🌟 功能特色

*   **儀表板**：即時查看本月代課統計、待聘節數與預估支出。
*   **代課單建立**：
    *   支援「鐘點費」與「日薪」兩種計算模式。
    *   日薪模式自動包含導師費計算。
    *   智慧課表輸入，可快速點選節次。
    *   歷史文號自動記憶與搜尋。
*   **待聘清單**：自動彙整尚未找到代課教師的課務，以顏色區分不同請假教師。
*   **教師管理**：管理校內/校外教師資料、薪級、證書狀態，並支援與 Google Sheet 雙向同步。
*   **報表輸出**：
    *   自動產生「印領清冊」（含自動換行格式）。
    *   自動產生「黏貼憑證」。
    *   支援批次匯出代課單。
*   **超鐘點計算**：針對校內教師計算每週超鐘點費用。

## 🚀 安裝與部署指南

本系統分為 **前端 (React)** 與 **後端 (Google Apps Script + Spreadsheet)** 兩部分。

### 第一步：後端設定 (Google Sheet & GAS)

1.  **建立 Google Sheet**：
    *   在您的 Google Drive 建立一個新的試算表，命名為「代課薪資管理系統資料庫」。
    *   記下網址中的 Spreadsheet ID (例如：`/d/1aBcDeFg.../edit` 中的 `1aBcDeFg...`)。

2.  **開啟 Apps Script**：
    *   在試算表中，點選 `擴充功能` > `Apps Script`。

3.  **複製程式碼**：
    *   將專案中 `gas/` 資料夾內**所有** `.gs` 檔依檔名一對一複製到 Apps Script（勿漏檔；檔名需與編輯器內一致）。

4.  **初始化系統**：
    *   開啟 `AppConfig.gs`，將 `SPREADSHEET_ID` 填入您的試算表 ID。
    *   開啟 `Setup.gs`，選擇 `runQuickSetup` 函式並執行。
    *   **注意**：執行時需授予權限。此步驟會自動在試算表中建立所需的工作表（原始紀錄、教師資料庫、薪級表、範本等）。

5.  **部署為 Web App**：
    *   點選右上角 `部署` > `新增部署作業`。
    *   **類型**：選取 `網頁應用程式`。
    *   **執行身分**：`我 (Me)`。
    *   **誰可以存取**：`任何人 (Anyone)` **(重要：必須選任何人，前端才能連線)**。
    *   部署後，複製產生的 **Web App URL** (以 `/exec` 結尾)。

### 第二步：前端設定 (React)

1.  **安裝依賴**：
    ```bash
    npm install
    ```

2.  **設定連線**：
    *   開啟專案根目錄的 `config.ts`。
    *   將 `GAS_WEB_APP_URL` 修改為您剛剛複製的 Web App URL。
    *   或者，您也可以在系統首次啟動時，於登入畫面輸入網址。

3.  **啟動開發伺服器**：
    ```bash
    npm run dev
    # 或
    npm start
    ```

## 📖 操作說明

### 1. 系統初始化
*   首次進入系統，建議先至 **「教師管理」** 頁面。
*   點擊 **「同步至雲端」** 確保本地端與 Google Sheet 資料一致。
*   若 Google Sheet 是空的，請手動新增教師資料，或要求從 Sheet 載入。
*   **注意**：請確保「薪級級距表」Sheet 內已有資料（執行 `runQuickSetup` 會自動填入預設值）。

### 2. 建立代課單 (核心功能)
1.  進入 **「新增代課單」**。
2.  **基本資料**：
    *   選擇「請假教師」。
    *   選擇「假別」。
    *   輸入或選擇「公文文號」（系統會自動搜尋歷史文號）。
3.  **課表輸入**：
    *   設定「科目」、「班級」。
    *   選擇「代課教師」（若尚未確定可選「待聘」）。
    *   選擇「支薪方式」：
        *   **鐘點費**：點擊課表上的格子，單節單節新增。
        *   **日薪**：選擇後，點擊任一格子，系統會自動填滿該日所有時段，並標記為日薪（含導師費計算）。
4.  點擊 **「儲存代課單資料」**。

### 3. 待聘處理
*   進入 **「待聘清單」**。
*   系統會列出所有尚未指定代課教師的課程。
*   點擊 **「前往排課」** 可直接跳轉回代課單編輯頁面進行人員指派。

### 4. 產生報表 (月底結算)
*   進入 **「代課清冊/憑證」**。
*   上方選擇月份。
*   點擊 **「同步更新」** 將所有紀錄寫入 Google Sheet 的「原始紀錄資料庫」。
*   點擊 **「匯出清冊/憑證」**：
    *   系統會呼叫後端，在 Google Drive 產生該月份的 Excel 檔案。
    *   包含「印領清冊」與「黏貼憑證」。
    *   **特色**：假別欄位會自動換行，例如 `公付\n(公假)`，符合行政需求。

### 5. 超鐘點計算
*   進入 **「超鐘點計算」**。
*   系統自動帶出校內教師。
*   **基本節數**：系統依據職別（導師/科任/組長/主任）預設基本節數，可手動修改。
*   **實際排課**：輸入每週實際節數。
*   **特殊調整**：輸入需扣除或增加的總節數（例如畢業班課程結束）。
*   系統會自動計算應發金額（無條件進位）。

## ⚠️ 注意事項

1.  **資料同步**：本系統採「本地優先」策略，操作後請記得點擊「同步」按鈕將資料存回 Google Sheet。
2.  **權限安全**：由於 Google Apps Script 部署為「任何人」可存取，請勿將 Web App URL 洩漏給無關人員。
3.  **瀏覽器快取**：若更新了 GAS 後端程式碼，建議重新整理網頁以確保連線正常。
