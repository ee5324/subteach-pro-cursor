## 清冊舊版備份（避免回不去）

為避免調整清冊（GAS 匯出 / 前端顯示）後發生錯誤無法回復，本專案已建立一個 Git tag 作為「舊版快照」。

- **備份 tag**：`ledger-backup-20260319`

此 tag 代表當時「清冊/憑證」相關邏輯的完整狀態（包含前端與 GAS 程式碼）。  
若未來需要回到舊版，請依以下方式操作。

---

## 如何切回舊版（檢視/比對）

```bash
git fetch --tags
git checkout ledger-backup-20260319
```

> 注意：checkout 到 tag 會進入「detached HEAD」狀態，只適合檢視與比對。

---

## 如何從舊版開新分支繼續修改

```bash
git fetch --tags
git checkout -b restore-ledger ledger-backup-20260319
```

之後即可在 `restore-ledger` 分支上做修正或 cherry-pick 需要的 commit。
