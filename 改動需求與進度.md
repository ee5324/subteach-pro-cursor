## 二、家長會清冊 — 完整規格彙整

> **命名**：家委會清冊與家長會清冊為**同一份**清冊，統一稱**家長會**。檔名、標題、介面皆用「家長會」。

### 2.1 納入規則（同一份清冊，兩種來源）

| 情況 | 條件 | 處理方式 |
|------|------|----------|
| **情況一** | 假別為「**公派(家長會)**」 | **整筆**請假／代課（鐘點／日薪／半日薪皆可）完整納入家長會清冊。 |
| **情況二** | 支薪為「**半日薪**」且（「**家委支出**」打勾 **或** 假別為「**身心假**」） | **僅導師費**納入家長會清冊；**薪水本身**仍依假別入各假別清冊（公假→公假清冊、身心假→身心假清冊等）。 |

- 情況一、情況二都寫入**同一份**家長會清冊（每月一檔）。情況二在家長會清冊中只列導師費請款，不重複列整筆薪資。

### 2.2 排除規則

- **半日薪 ＋ 假別「自理」**：只納入**自理清冊**，**不**納入家長會清冊。
- **身心假 ＋ 半日薪**：導師費納入家長會清冊（無須勾選家委支出）。

### 2.3 實作時需補齊

- **假別**：新增「**公派(家長會)**」選項（如 `LeaveType` 或假別選單）。
- **家委支出**：新增「**家委支出**」欄位（打勾＝該筆導師費由家長會支出）；可放在請假單或代課明細，實作時決定層級。

### 2.4 範本與產出

- **範本**：Google 試算表工作表「**家長會清冊範本**」；複製範本、填資料、產出（與其他印領清冊相同邏輯）。
- **檔名**：`民國年年年+月月_家長會_印領清冊`。
- **標題**：例「加昌國小OOO年O月代課教師印領清冊~~【級科任教師】公假家長會」。

### 2.5 欄位對應（A2～N2）

| 欄 | 表頭 | 填入內容 |
|----|------|----------|
| A | 代課日期 | 代課日期彙總 |
| B | 代課老師 | 代課教師姓名 |
| C | 薪級 | 薪級＋(有證/無證) |
| D | 表頭動態 | 見 2.6 |
| E | 代課天數 | 代課天數（半日 0.5） |
| F | 代課節數 | 代課節數 |
| G | 代課鐘點費 | 鐘點費小計 |
| H | 請假人姓名 | 請假人 |
| I | 假別 | 假別 |
| J | 請假事由 | 請假事由 |
| K | 備註 | 備註（如半日0節） |
| L | 代導師日數 (半日導師費家長會請款) | 代導師日數（半日 0.5） |
| M | 表頭動態 | 見 2.6 |
| N | 應發金額 | 應發金額 |

### 2.6 表頭動態寫入（產出時寫入）

- **D2**：`日薪` + 換行(Alt+Enter) + `(OO日)`，OO＝當月日數（28/29/30/31）。
- **M2**：`導師費(M月OOO元)`，M＝月份（1～12），OOO＝導師費每月單價（如 4000，可改為設定值）。
