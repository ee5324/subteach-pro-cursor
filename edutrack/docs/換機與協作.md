# 換電腦與協作說明

換台電腦或與他人協作時，透過 **pull** 即可取得最新程式與 Firestore 規則；資料存於 Firebase，登入同一專案即可延續紀錄。

---

## 另一台電腦要做的事

1. **取得程式**：`git clone` 或 `git pull`
2. **安裝依賴**：`npm install`
3. **環境檔**：複製 `.env.example` 為 `.env`，填入 Firebase、GAS 等（每台電腦自建，勿提交）
4. **執行**：`npm run dev`（預設 http://localhost:5180）

## 紀錄與規則如何同步

| 項目 | 說明 |
|------|------|
| **程式與規則** | 透過 Git **push / pull** 同步；`firestore.rules` 在 repo 內，pull 即取得最新規則。 |
| **Firestore 資料** | 課程、學生、頒獎、廠商、事項列檔、待辦、**學生語言選修登錄**、考卷存檔等皆在 Firebase，登入同一專案即可看到，無需搬檔。 |
| **Firebase Console** | 若在 Console 改過規則，請將規則同步回專案的 `firestore.rules` 並 push，以利他台與協作者一致。 |
